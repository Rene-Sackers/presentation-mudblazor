## Manual installation

https://mudblazor.com/getting-started/installation#manual-install-add-components

```
dotnet add package MudBlazor
```

```
// _Imports.razor

@using MudBlazor
```

```
// App.razor

<link href="https://fonts.googleapis.com/css?family=Roboto:300,400,500,700&display=swap" rel="stylesheet" />
<link href="@Assets["_content/MudBlazor/MudBlazor.min.css"]" rel="stylesheet" />

// ...

<script src="@Assets["_content/MudBlazor/MudBlazor.min.js"]"></script>
```

```
// Program.cs

using MudBlazor.Services;

builder.Services.AddMudServices();
```

```
// MainLayout.razor

@* Required *@
<MudThemeProvider />
<MudPopoverProvider />

@* Needed for dialogs *@
<MudDialogProvider />

@* Needed for snackbars *@
<MudSnackbarProvider />
```

**Run it!**

## Basic Layout

### Add an app bar and drawer

```
<MudLayout>  
    <MudAppBar>  
       <MudIconButton Icon="@Icons.Material.Filled.Menu" Color="Color.Inherit" Edge="Edge.Start" Class="mr-2"/>
       My Application
    </MudAppBar>
    <MudDrawer>
    </MudDrawer>
    <MudMainContent>
	  @Body
    </MudMainContent>
</MudLayout>
```

## Add open/close

```
OnClick="@((e) => DrawerToggle())"

@code {  
    bool _drawerOpen = true;  
  
    void DrawerToggle()  
    {  
       _drawerOpen = !_drawerOpen;  
    }  
}
```

## Add nav menu

```
<MudDrawer @bind-Open="@_drawerOpen">  
    <MudNavMenu>  
       <MudNavLink Href="/" Match="NavLinkMatch.All">Home</MudNavLink>  
       <MudNavLink Href="/HelloWorld" Match="NavLinkMatch.Prefix">Hello world</MudNavLink>
	</MudNavMenu>
</MudDrawer>
```

Create new page `HelloWorld.razor`
