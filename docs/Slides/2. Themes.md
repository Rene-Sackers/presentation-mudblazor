# Themes, styling, CSS

## Create a new theme, set primary color

### Custom theme
```
// MainLayour.razor

// Add new theme
private readonly MudTheme _myTheme = new()
{
	PaletteLight = new()
	{
		Primary = new MudColor(20, 180, 230, 255),
		AppbarBackground = new MudColor(20, 180, 230, 255)
	}
};

// Set theme on provider
<MudThemeProvider Theme="@_myTheme"/>

// Test it out!
<MudButton Variant="Variant.Filled" Color="Color.Primary">Hello .NET Zuid!</MudButton>
```

See https://mudblazor.com/customization/default-theme#palette for the entire pallette.

### Dark theme
```
<MudThemeProvider IsDarkMode="true"/>
```

User preference theme
```
<MudThemeProvider @ref="_mudThemeProvider" Theme="_myTheme" @bind-IsDarkMode="_isDarkMode"/>

private bool _isDarkMode;
private MudThemeProvider _mudThemeProvider;

protected override async Task OnAfterRenderAsync(bool firstRender)
{
	if (firstRender)
	{
		_isDarkMode = await _mudThemeProvider.GetSystemDarkModeAsync();
		StateHasChanged();
	}
}
```

## CSS/Flexbox classes

### Margin/padding

```
ma-4 -> margin ALL 4
my-4 -> margin Y-axis 4
mx-4 -> margin X-axis 4
mt-4, mr-4, mb-4, ml-4 -> margin top|right|bottom|left

pa, px, py, pt, pr, pb, pl -> padding all|x|y||top|right|bottom|left
```

```
d-flex
flex-column
flex-row
flex-grow-1

align-items-center
align-content-center

```
